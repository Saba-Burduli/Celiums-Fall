# Asset Credits

## Original Celium sprite set

The original 16-bit-style character sheet in `assets/original/` was created specifically for Celium's Fall. The SVG is the editable source; run `swift scripts/render_original_assets.swift` from the repository root to regenerate the PNG.

## Curated sprite set

`assets/curated/tiny-dungeon.png` is adapted directly from **Tiny Dungeon 1.0** by [Kenney](https://kenney.nl/assets/tiny-dungeon). The pack is released under Creative Commons Zero (CC0). The original license text is stored beside the asset as `assets/curated/LICENSE.txt`.

## Gothic platformer art

The primary side-view presentation uses [GothicVania Church Pack](https://opengameart.org/content/gothicvania-church-pack) and [Gothicvania Magic Pack 9](https://opengameart.org/content/gothicvania-magic-pack-9) by Ansimuz. Both packs are CC0. The repository includes only the environment, character, enemy, and spell files used by the game; original license texts are stored in `assets/gothic/`.

## Pixel font

The UI uses `KenPixel Square` from [Kenney Fonts](https://opengameart.org/content/kenney-fonts), released under CC0. Its license is stored in `assets/fonts/LICENSE.txt`.
